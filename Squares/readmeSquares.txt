Squares
Let's have N rows and M columns of squares. Each square can be colored in either black or white. Write a program which finds the number of colorings where there are no two black squares next to each other.

Input
Input is read from the console
N is given on the first line
M is given on the second line
Output
Output should be printed on the console
Constraints
1 <= N, M <= 100
1 <= Min(N, M) <= 10
Time limit: 0.2s
Memory limit: 32 MB
